这个关于我自己的人生操作系统的设计，我是极其骄傲的，是一个极其完备的大一统系统，统一了所有，极其完备，逻辑极其闭环，在发布后的第二天也就是今天，我又想到了一些关于宇宙中冥冥中的真理、天道。

这个系统牛逼就牛逼在闭环了我从小到大就一直梦想、真正想做的事情，当然，这里面最重要的之一就是探索世界的未知，我想无数人都曾梦想过当科学家，成为牛顿、爱因斯坦，成为那个在海边捡贝壳、窥探宇宙真理的人。（参考刘慈欣《朝闻道》）

然而，窥探宇宙的真理是一个逆熵增的过程，这是一个触碰“天机不可泄漏”的事情，（依然参考刘慈欣《朝闻道》《2001太空漫游》），人类的智慧的诞生从猿猴仰望星空和使用工具那一刻开始，一方面是背负了对未知的无尽的探索欲、来自知识的curse诅咒，另一方面是对生存的效率的不停追求和繁衍生存本能的不断驱使。

我真正想做的事自然是推动人类的科学进步和发展，但这毕竟是需要耗费大量精力（对人类来说就是科学发展到今天继续进行实验需要投入天量的资源）和强烈依赖天赋的事情，同时受限于我的发展轨迹和天赋，我必然要先解决生存问题。（对于人类社会国别竞争来说，基础科学研究本就是一体两面，既有人类发展使命，又是为了生存与他国竞争抢占先机的并非纯粹的纯利益驱使，依旧是一个一体两面、对立统一的天道）

包括我想扣篮。

这些真正想做的事情在我接下来这二三四十年、生命机能的顶峰中固然需要去追求，但这终将会有一个结束，归于平静，吾生也有涯知也无涯，此乃天道。

我想我会尽量依据这套完备的系统去践行，希望在我行将结束我人生之际，能在与这个世界发生足够多的交互、多少完成一些自己想做的事情或是什么也没有做也无所谓之后，重新回归这个宇宙。

这是在操作系统之上的最高闭环，也就是zhuOS里面写到的：宇宙的无意义或悲观，但其实用悲观来形容是不正确的，因为宇宙本就没有乐观悲观之分，天地不仁，宇宙本身高悬于万物之上，不带有任何色彩，无所谓有无是非，空间中的任何一点只是静静地发生、消失。
